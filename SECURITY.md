# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

Please open a **private** GitHub security advisory on
[vibe-code-commit/model-lock](https://github.com/vibe-code-commit/model-lock)
when available, or open a GitHub Issue without including exploit details.

Do not email personal addresses. This project is operated semi-anonymously under
**Rabbott LLC** copyright.

## Scope

ModelLock does not call provider inference APIs and does not collect telemetry.
Reports related to lockfile secret scrubbing, HTML sanitization, network
timeouts/size limits, and Action permission defaults are especially welcome.
